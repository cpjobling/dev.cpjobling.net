Welcome to my Nikola-driven dev blog¶

I am building this blog following the instructions given by Jaakko Luttinen. I am using the static site generator Nikola because it supports jupyter notebooks as well as markdown and restructured text (RST) formatters.

The blog is hosted using GitHub pages at cpjobling.github.io/dev.cpjobling.net as well as on Reclaim Hosting at dev.cpjobling.net.

For archival purposes I have moved over some posts that were originally developed using Octopress and hosted on cpjobling.github.io.
